# Tokyo Spring 2027

Cherry blossom timing is a gamble — going the last week of March because peak bloom in Tokyo is usually around March 28-April 1. If we're early, we still hit a few blooms. If we're late, we get full peak.

## Vibe

Mixed city + day trip. Two slow mornings, two long days, one full-day Hakone run. Hotel in Shinjuku so we can walk to dinner every night.

- Light jacket (March in Tokyo is 8-15°C)
- Walking shoes — 18-25k steps a day
- Pocket WiFi or eSIM
- Cash (¥30k starting, top up at 7-Eleven ATM)
- Onsen kit (small towel, modesty for tattoo if needed)
